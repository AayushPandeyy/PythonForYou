Boolean Data Type

What are Booleans?

A boolean expression (or logical expression) evaluates to one of two states true or false. Python provides the boolean type that can be either set to False or True. Many functions and operations returns boolean objects. The not keyword can also be used to inverse a boolean type. >>> not True False.
True returns 1 as an integer value and False returns 0 as an inter value.
